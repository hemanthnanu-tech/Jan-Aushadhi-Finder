## Overview
**Jan-Aushadhi Finder** is a specialized Android application designed to make healthcare more accessible and affordable. It allows users to quickly search for branded medicines and instantly discover their official, generic alternatives available under the Jan-Aushadhi scheme. By providing clear side-by-side price comparisons, the app empowers users to make informed, cost-effective healthcare decisions.

## ✨ Key Features
* **Smart Search:** Quickly look up any branded medicine (e.g., Dolo, Cipla products).
* **Instant Alternatives:** View exact generic counterparts available at Jan-Aushadhi Kendras.
* **Price Comparison:** Clear, side-by-side display of branded vs. generic prices.
* **Savings Calculator:** Automatically calculates and displays the exact amount and percentage saved.
* **Clean, Modern UI:** A highly intuitive, distraction-free interface built for ease of use.

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/hemanthnanu-tech/Jan-Aushadhi-Finder.git](https://github.com/hemanthnanu-tech/Jan-Aushadhi-Finder.git)
2. **Open Android Studio.**

3. **Select File > Open and navigate to the cloned project directory.**

4. **Allow Gradle to sync and build the project dependencies.**

5. **Click Run to launch the app on an emulator or physical Android device.**
